================================================
脚注
================================================

脚注分为两个部分: 

* 正文中的脚注标号
* 页底的脚注说明

定义脚注有多种方式

- `\[#\]\_`: 自动编号的脚注, 按照在文中出现的顺序与引文内的顺序对应
- `\[#note\]\_`: 带标签的脚注, 在任意位置通过该形式
- `\[citation\]`: 引文形式, 文字内容直接显示在正文中, 而非显示编号.
