SQLAlchemy
================================================

`SQLAlchemy <https://www.sqlalchemy.org>`_, 用于实现ORM的Python框架, 可以直接使用也可以与Flask集成.

支持常见的数据库, Mysql, SQLServer, sqlite等.

安装

.. code-block:: bash
    
    pip install SQLAlchemy
