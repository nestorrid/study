常用插件
================================================

`pytest官方提供的插件列表 <https://docs.pytest.org/en/7.4.x/reference/plugin_list.html#plugin-list>`_

几款比较常用的插件:

allure-pytest, pytest-html
    生成测试报告

pytest-xdist
    多线程支持

pytest-ordering
    控制用例的执行顺序

pytest-rerunfailures
    失败用例重跑

pytest-base-url
    管理基础路径
