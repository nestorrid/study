# 3. 常用标记

## 指定顺序 - run

默认情况下, pytest会按照模块名, 用例名, 从上而下顺序执行.

可以通过`run`装饰器修改特定用例的执行顺序:

```python
@python.mark.run(order=n)
```

---

## 预期失败 - xfail

有些用例会在特定情况下失败, 属于可预计且允许的失败, 此时可以添加`xfail`标记.

```python
@pytest.mark.xfail(reason="除数为0")
def test_divide_by_zero():
    print(1 / 0)
```
